Ladies Learning Code is a Toronto-based not-for-profit organization that runs workshops for women (and men, and girls and boys) who want to learn beginner-friendly computer programming and other technical skills in a social and collaborative way. Today we also have chapters across Canada and programs for youth called Girls Learning Code and Kids Learning Code. Oh, and we're just getting started.
Want to get involved? Contact us.
